* euler

  My attempts at learning clojure by solving the Project Euler problems.

** Usage

   #+BEGIN_SRC clojure
   user> (use 'euler.core)
   ;=> nil
   user> (euler.core/problem-001 10)
   ;=> 23
   #+END_SRC
