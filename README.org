* euler

  Learn [[http://clojure.org/][Clojure]] via [[http://projecteuler.net/][Project Euler]]

** Usage

   #+BEGIN_SRC clojure
   user> (use 'euler.core)
   ;=> nil
   user> (euler.core/problem-001 10)
   ;=> 23
   #+END_SRC
